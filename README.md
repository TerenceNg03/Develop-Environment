# Develop-Environment
Setup script for vim &amp; zsh

Include a docker file and config for zsh & vim.
Note that `install.sh` **does not need root access** as it uses `homebrew`.

To install run 
```sh
./install.sh
```
Or build a docker image run 
```
docker build .
```

## Setup for font &amp; theme
This configuration uses `Gruvbox-light` theme. It will install a patched powerline font named `Droid Sans Nerd Font`. Make sure you configure your terminal to use that font in order to avoid broken texts.
